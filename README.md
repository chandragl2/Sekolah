# 🏫 Sistem Informasi Sekolah

Aplikasi berbasis web untuk mengelola data **siswa, guru, mata pelajaran, dan ujian**.  
Dibangun menggunakan **PHP Native**, **MySQL**, dan **Bootstrap**.

---

## 🔧 Tech Stack
- PHP Native
- MySQL
- Bootstrap 5
- jQuery

---

## ✨ Fitur Utama
- Login Admin
- CRUD Data Siswa
- CRUD Data Guru
- CRUD Mata Pelajaran
- CRUD Ujian
- Upload Foto Siswa
- Ganti Password Admin
